# Audit Evidence Schemas

Built-in schemas:

- `internal-secure-release`
- `startup-production-readiness`
- `soc2-security-mapping`
- `iso27001-secure-sdlc-mapping`
- `ai-appsec-readiness`
- `crypto-fintech-release-readiness`

Schemas are checklist and evidence mapping templates. They must include a disclaimer and must not claim certification or compliance.

Commands:

```bash
vibe audit-schemas list
vibe audit-schemas show internal-secure-release
vibe audit-schemas validate
vibe audit-schemas install-defaults
```
