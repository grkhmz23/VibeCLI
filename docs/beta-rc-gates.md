# Beta RC Gates

`vibe beta-rc` aggregates local dogfood, security, package, docs, performance, scanner, warning, and backlog reports.

Commands:

- `vibe beta-rc`
- `vibe beta-rc --strict`
- `vibe beta-rc --channel private-beta`

Verdicts are scoped to beta: `beta_rc_ready`, `ready_with_accepted_warnings`, or `blocked`.

This is not a production-readiness or compliance certification claim.
