# Recovery Guidance

Phase 15 recovery guidance is best-effort only. VibeCLI cannot guarantee recovery after local deletion.

After disposal, check:

- VCS history
- local backups
- filesystem backups
- CI artifacts
- evidence archives
- handoff bundles
- release packets
- evidence exports

VibeCLI does not delete source code, archives, private keys, `.env` files, or remote evidence in Phase 15.
