# Team Handoff

Phase 7 produces redacted handoff artifacts for code review.

```bash
vibe handoff <run-id>
vibe handoff <run-id> --verify
vibe handoff <run-id> --zip
```

The bundle includes a summary, PR body, reviewer checklist, policy exceptions, approval notes, and a handoff manifest. It contains only redacted handoff files, not the full run directory.
